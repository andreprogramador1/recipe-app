# Recipe app

## Pré-requisitos

- Navegador (Chrome, FireFox, Edge, etc...)

## Executar

1. Clonar repositório em algum diretório.
2. Abrir o arquivo index.html.
3. Escrever nome de receita
4. Clicar na imagem para ver igredientes
